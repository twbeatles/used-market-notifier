# 🥕 중고거래 알리미 (Used Market Notifier)

당근마켓, 번개장터, 중고나라에서 원하는 물건을 실시간으로 검색하고 알림을 받아보세요.
현대적인 UI와 강력한 필터링 기능으로 중고거래를 더욱 스마트하게 도와줍니다.

## ✨ 주요 기능

### 🔍 강력한 모니터링
*   **다중 플랫폼 지원**: 당근마켓, 번개장터, 중고나라 동시 모니터링
*   **병렬 검색**: 멀티스레딩/비동기 방식을 사용한 빠른 검색 속도
*   **중복 필터링**: 단순 ID 비교를 넘어선 Fuzzy Matching으로 재업로드(끌어올리기) 상품 필터링
*   **판매자 차단**: 악성 판매자나 업자 계정 필터링

### 🎯 스마트한 키워드 관리
*   **정밀 필터링**: 최소/최대 가격, 제외 키워드, 지역 설정 (당근마켓)
*   **개별 스케줄**: 키워드별 검색 주기 개별 설정 가능 (예: 고가 상품 1분, 저가 상품 60분 간격)
*   **손쉬운 관리**: 드래그 앤 드롭 정렬, 원클릭 활성화/비활성화 (구현 중)

### 📊 데이터 시각화 & 관리
*   **대시보드**: 실시간 검색 통계, 플랫폼별 분포, 일별 추이 그래프
*   **가격 변동 추적**: 등록된 상품의 가격 하락 감지 및 알림
*   **즐겨찾기**: 관심 상품 저장 및 메모, 목표 가격 설정 시 도달 알림
*   **알림 내역**: 발송된 모든 알림 히스토리 조회 및 필터링
*   **데이터 내보내기**: 검색 결과 및 알림 내역을 Excel/CSV 파일로 저장

### 🎨 사용자 편의성
*   **현대적인 UI**: Tokyo Night 테마 기반의 세련된 다크/라이트 모드 지원
*   **다양한 알림**: 텔레그램(이미지 포함), 디스코드, 슬랙 지원
*   **안전한 탐색**: 상품 링크 열기 전 확인 문구 표시 옵션
*   **백그라운드 실행**: 시스템 트레이 모드 지원으로 방해 없이 동작

## 🛠️ 설치 및 실행

### 요구 사항
*   Windows 10/11
*   Python 3.10 이상
*   Chrome 브라우저 (Google Chrome)

### 1. 설치
```bash
git clone https://github.com/yourusername/used-market-notifier.git
cd used-market-notifier
pip install -r requirements.txt
```

### 2. 실행
```bash
python main.py
```

### 3. 실행 파일 빌드 (.exe)
```bash
pip install pyinstaller
pyinstaller used_market_notifier.spec
```
빌드 완료 후 `dist/UsedMarketNotifier.exe` 실행

## ⚙️ 설정 가이드

1.  **키워드 설정**: 키워드 추가 탭에서 검색어, 가격 조건, 제외 키워드 입력
2.  **알림 설정**: 
    - **텔레그램**: 봇 토큰 및 Chat ID 입력 (테스트 발송 권장)
    - **디스코드/슬랙**: 웹훅 URL 입력
3.  **일반 설정**:
    - 테마 (다크/라이트/시스템)
    - 시작 프로그램 등록
    - 링크 오프닝 확인창 활성화
